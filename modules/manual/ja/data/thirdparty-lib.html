サードパーティ製ライブラリ

Loggixは基本コンセプトとしてユーザーが独自にPEARなどの外部ライブラリを簡単に導入して拡張して行けることを目指しています。また、システム自体もいくつかの外部ライブラリに依存して構築されています。

Loggixに標準で組み込まれているサードパーティ製の外部ライブラリは以下の通りです。

PEAR

  • PEAR :: Package :: Text_Highlighter

JavaScript

  • jQuery: The Write Less, Do More, JavaScript Library
  • ColorBox - customizable lightbox plugin for jQuery

PHPプラグイン

  • PHP Markdown Extra
  • Visual and Audio PHP CAPTCHA Generation Class - Ed Eliot

アイコン

  • Tango Desktop Project

FAQ

Q : ユーザーが独自に他のサードバーティ製ライブラリを追加出来ますか？
A : はい、出来ます。 「/lib/php」に入れます。これらは自動的にロードされます。
