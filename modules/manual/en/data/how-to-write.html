What is Markdown?

Markdown is a text-to-HTML conversion tool for web writers. Markdown allows you to write using an easy-to-read, easy-to-write plain text format, then convert it to structurally valid XHTML (or HTML).

Daring Fireball

Loggix implements PHP version of Markdown which is available here : PHP Markdown Extra

Markdown is one of the best solutions for whom like to use xhtml tags and easy-to-read text mixed on web publishing.

Using (X)HTML tags

You can also use (X)HTML tags to markup specific text with markdown syntax.

Example :

To emphasize, you can use these classes :

  1. <em>
  2. <strong>
  3. <strong class="stronger">
  4. <strong class="very-strong">

These tags shown above can be used like this:

<em>EMPHASIZE</em>
<strong>STRONG</strong>
<strong class="stronger">STRONGER</strong>
<strong class="very-strong">VERY STRONG</strong>

...and will be rendered like this: ↓

EMPHASIZE STRONG STRONGER VERY STRONG

External link samples :

Use ex-ref class to define external references.

<a href="http://loggix.gotdns.org/"
title="Loggix Project Official Site"
class="ex-ref">Loggix Project</a>

will be rendered like this↓

Loggix Project

Quotations Examples :

Use <blockquote> when you quote. Use address tag to identify the resource.

<blockquote>
<p>Quote from Loggix Project</p>
<address>
<a href="http://loggix.gotdns.org/">Loggix Project</a>
</address>
</blockquote>

This will be renedered like this ↓

Quote from Loggix Project

Loggix Project

Computer expressons examples :

Use <kbd> to write typed text.

<kbd>shift</kbd>
<kbd>option</kbd>
<kbd>command</kbd>
+
<kbd class="important">S</kbd>

This will be rendered like this ↓

shiftoptioncommand+S
