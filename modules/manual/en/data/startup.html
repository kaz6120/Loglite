Install

STEP1

Upload all files in public_html or htdocs directory of your server.

  • plubic_html / htdocs
    • loggix

Set permissions of the directories and files shown below readable and rewritable.

  • loggix
    • data
      • resources(Destination of uploaded images and files)
      • loggix.sqlite3.db(SQLite3 database file)
      • loggix.sqlite.db(SQLite2 database file)

Loggix will read SQLite database files in order of:

  1. loggix.sqlite3.db(SQLite3 database file)
  2. loggix.sqlite.db(SQLite2 database file)

so just remove loggix.sqlite3.db if you want to use SQLite2.

STEP2

Open in your browser:

http://your-domain.com/path/to/loggix/admin/login.php

and access:

  • loggix
    • admin
      • login.php ← HERE!

STEP3

Log in as :

  • User : root
  • Password : root

STEP4

After you logged in, create a new user account and log out. Then log in again by using a new user account and then delete the original root user account for some security reasons.

And...That's it. You're ready to Loggix!

System prefereneces

Customizing

All view stuffs (such as HTML template files and CSS) are in "theme" directory. Please edit these files as you like with your favorite text editor to change and customize "View"
