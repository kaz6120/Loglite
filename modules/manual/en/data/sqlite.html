How to backup database

Here's how to backup SQlite database that Loggix use. Use CLI to execute the backup command.

Backup and restore

(1) Dump data to SQL file.

$ echo '.dump' | sqlite loggix.sqlite.db | cat > loggix.sqlite.sql

(2) Restore database from dumped SQL file.

$ sqlite loggix.sqlite.db < loggix.sqlite.sql

You can also optimize database file to execute these commands.
